This can be done as follows. Since \(b\neq 0\text{,}\) then \(\frac{|b|}{2}>0\text{.}\) Thus, by the definition of \(\lim_{n\rightarrow\infty}b_n=b\) , there exists \(N_1\) such that if \(n>N_1\text{,}\) then \(|b\mathopen|-|b_n|\leq\big|b-b_n\big|\lt \frac{|b|}{2}\text{.}\) Thus when \(n>N_1\text{,}\) \(\frac{|b|}{2}\lt |b_n|\) and so \(\frac{1}{|b_n|}\lt \frac{2}{|b|}\text{.}\) This says that for \(n>N_1\text{,}\) \(\frac{|b-b_n|}{|b_n||b|}\lt \frac{2}{|b|^2}|b-b_n|\text{.}\) We should be able to make this smaller than a given \(\eps>0\text{,}\) provided we make \(n\) large enough.
