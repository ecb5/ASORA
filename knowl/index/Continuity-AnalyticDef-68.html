Problem 8.1.20.

(a)

Use the definition of continuity to show that the function
\begin{equation*} D(x)= \begin{cases} x,\amp \text{ if } x\text{ is rational } \\ 0,\amp \text{ if } x\text{ is irrational } \end{cases} \end{equation*}
is continuous at \(0\text{.}\)

(b)

Let \(a\neq 0\text{.}\) Use the definition of continuity to show that \(D\) is not continuous at \(a\text{.}\)
Hint.
You might want to break this up into two cases where \(a\) is rational or irrational. Show that no choice of \(\delta>0\) will work for \(\eps=|\,a|\text{.}\) Note that Theorem 2.0.11 of Chapter 2 will probably help here.
