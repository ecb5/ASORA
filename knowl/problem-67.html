Problem 7.2.5.

Prove Theorem 7.2.4.
SCRAPWORK:
If we let \(\eps>0\text{,}\) then we want \(N\) so that if \(n>N\text{,}\) then \(\big|\left(a_n+b_n\right)-\left(a+b\right)\big|\lt \eps\text{.}\) We know that \(\limit{n}{\infty}{a_n}=a\) and \(\limit{n}{\infty}{b_n}=b\text{,}\) so we can make \(\big|a_n-a\big|\) and \(\big|b_n-b\big|\) as small as we wish, provided we make \(n\) large enough. Let’s go back to what we want, to see if we can close the gap between what we know and what we want.
\begin{equation*} \left|\left(a_n+b_n\right)-\left(a+b\right)\right|=\left|\left(a_n-a\right)+\left(b_n-b\right)\right|\leq\left|a_n-a\right|+\left|b_n-b\right| \end{equation*}
by the triangle inequality. To make this whole thing less than \(\eps\text{,}\) it makes sense to make each part less than \(\frac{\eps}{2}\text{.}\) it makes sense to make each part less than \(\frac{\eps}{2}\text{.}\) Fortunately, we can do that as the definitions of \(\limit{n}{\infty}{a_n}=a\) and \(\limit{n}{\infty}{b_n}=b\) allow us to make \(\big|a_n-a\big|\) and \(\big|b_n-b\big|\) arbitrarily small. Specifically, since \(\limit{n}{\infty}{a_n}=a\text{,}\) there exists an \(N_1\) such that if \(n>N_1\) then \(\big|a_n-a\big|\lt \frac{\eps}{2}\text{.}\) Also since \(\limit{n}{\infty}{b_n}=b\text{,}\) there exists an \(N_2\) such that if \(n>N_2\) then \(\big|b_n-b\big|\lt \frac{\eps}{2}\text{.}\) Since we want both of these to occur, it makes sense to let \(N=\)max\(\left(N_1,N_2\right)\text{.}\) This should be the \(N\) that we seek.
