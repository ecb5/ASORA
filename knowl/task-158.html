Task 13.2.2.a.

Prove that the following must hold in any linearly ordered number field.
  1. \(0\lt x\) if and only if \(-x\lt 0\text{.}\)
  2. If \(x\lt y\) and \(z\lt 0\) then \(y\cdot z\lt x\cdot z\text{.}\)
  3. For all \(x\neq 0\text{,}\) \(0\lt x^2\text{.}\)
  4. \(0\lt 1\text{.}\)
