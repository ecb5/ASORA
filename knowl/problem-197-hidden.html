Determine the derived set, \(S^\prime\text{,}\) of each of the following sets.

  1. \(\displaystyle S=\left\{\frac11, \frac12, \frac13, \ldots\right\}\)

  2. \(\displaystyle S=\left\{0,\frac11, \frac12, \frac13, \ldots\right\}\)

  3. \(\displaystyle S=(0,1]\)

  4. \(\displaystyle S=\left[\left.0,1/2\right)\right.\cup\left.\left(1/2,1\right.\right]\)

  5. \(\displaystyle S=\QQ\)

  6. \(\displaystyle S=\RR-\QQ\)

  7. \(\displaystyle S=\ZZ\)

  8. Any finite set \(S\text{.}\)
