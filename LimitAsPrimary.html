    Section 7.2 The Limit as a Primary Tool

    As you‚Äôve seen from the previous sections, the formal definition of the convergence of a sequence is meant to capture rigorously our intuitive understanding of convergence. However, the definition itself is an unwieldy tool. If only there was a way to be rigorous without having to run back to the definition each time. Fortunately, there is a way. If we can use the definition to prove some general rules about limits then we could use these rules whenever they applied and be assured that everything was still rigorous. A number of these should look familiar from calculus.

    Problem 7.2.1.

    Let \(\left(c\right)_{n=1}^\infty=(c,c,c,\ldots)\) be a constant sequence. Show that \(\lim_{n\rightarrow\infty}c=c\text{.}\)
    In proving the familiar limit theorems, the following will prove to be a very useful tool.

    Lemma 7.2.2.

    Triangle Inequality:
    Let \(a\) and \(b\) be real numbers. Then
    \begin{equation*} |a+b|\leq|a|+|b|\text{.} \end{equation*}
    Reverse Triangle Inequality:
    Let \(a\) and \(b\) be real numbers. Then
    \begin{equation*} |a|-|b|\leq\abs{a-b} {} \end{equation*}

    Problem 7.2.3.

    (a)

    Prove Lemma¬†7.2.2.
    Hint.
    For the Reverse Triangle Inequality, consider \(|a|=|a-b+b|\text{.}\)

    (b)

    Show \(||a|-|b||\leq|a-b|\text{.}\)
    Hint.
    You want to show \(|a|-|b|\leq|a-b|\) and \(-(|a|-|b|)\leq|a-b|\text{.}\)

    Theorem 7.2.4.

    If \(\displaystyle\lim_{n\rightarrow\infty}a_n=a\) and \(\displaystyle\lim_{n\rightarrow\infty}b_n=b\text{,}\) then \(\displaystyle\lim_{n\rightarrow\infty}\left(a_n+b_n\right)=a+b\text{.}\)
    We will often informally state this theorem as ‚Äúthe limit of a sum is the sum of the limits.‚Äù However, to be absolutely precise, what it says is that if we already know that two sequences converge, then the sequence formed by summing the corresponding terms of those two sequences will converge and, in fact, converge to the sum of those individual limits. We‚Äôll provide the scrapwork for the proof of this and leave the formal write-up as an exercise. Note the use of the triangle inequality in the proof.

    Problem 7.2.5.

    Prove Theorem¬†7.2.4.
    SCRAPWORK:
    If we let \(\eps>0\text{,}\) then we want \(N\) so that if \(n>N\text{,}\) then \(\big|\left(a_n+b_n\right)-\left(a+b\right)\big|\lt \eps\text{.}\) We know that \(\limit{n}{\infty}{a_n}=a\) and \(\limit{n}{\infty}{b_n}=b\text{,}\) so we can make \(\big|a_n-a\big|\) and \(\big|b_n-b\big|\) as small as we wish, provided we make \(n\) large enough. Let‚Äôs go back to what we want, to see if we can close the gap between what we know and what we want.
    \begin{equation*} \left|\left(a_n+b_n\right)-\left(a+b\right)\right|=\left|\left(a_n-a\right)+\left(b_n-b\right)\right|\leq\left|a_n-a\right|+\left|b_n-b\right| \end{equation*}
    by the triangle inequality. To make this whole thing less than \(\eps\text{,}\) it makes sense to make each part less than \(\frac{\eps}{2}\text{.}\) it makes sense to make each part less than \(\frac{\eps}{2}\text{.}\) Fortunately, we can do that as the definitions of \(\limit{n}{\infty}{a_n}=a\) and \(\limit{n}{\infty}{b_n}=b\) allow us to make \(\big|a_n-a\big|\) and \(\big|b_n-b\big|\) arbitrarily small. Specifically, since \(\limit{n}{\infty}{a_n}=a\text{,}\) there exists an \(N_1\) such that if \(n>N_1\) then \(\big|a_n-a\big|\lt \frac{\eps}{2}\text{.}\) Also since \(\limit{n}{\infty}{b_n}=b\text{,}\) there exists an \(N_2\) such that if \(n>N_2\) then \(\big|b_n-b\big|\lt \frac{\eps}{2}\text{.}\) Since we want both of these to occur, it makes sense to let \(N=\)max\(\left(N_1,N_2\right)\text{.}\) This should be the \(N\) that we seek.

    Theorem 7.2.6.

    If \(\displaystyle\lim_{n\rightarrow\infty}a_n=a\) and \(\displaystyle\lim_{n\rightarrow\infty}b_n=b\text{,}\) then \(\displaystyle\lim_{n\rightarrow\infty}\left(a_n b_n\right)=a b\text{.}\)
    SCRAPWORK: Given \(\eps>0\text{,}\) we want \(N\) so that if \(n>N\text{,}\) then \(\big|a_n b_n-a b\big|\lt \eps\text{.}\) One of the standard tricks in analysis is to ‚Äúuncancel.‚Äù In this case we will subtract and add a convenient term. Normally these would ‚Äúcancel out,‚Äù which is why we say that we will uncancel to put them back in. You already saw an example of this in proving the Reverse Triangle Inequality (Problem¬†7.2.3). In the present case, consider
    \begin{align*} \left|a_n b_n-a b\right|\amp =\left|a_n b_n-a_n b+a_n b-a b\right|\\ \amp \leq\left|a_n b_n-a_n b\right|+\left|a_n b-a b\right|\\ \amp =\left|a_n\right|\left|b_n-b\right|+\left|b\right|\left|a_n-a\right|\text{.} \end{align*}
    We can make this whole thing less than \(\eps\text{,}\) provided we make each term in the sum less than \(\frac{\eps}{2}\text{.}\) We can make \(\big|b\big|\big|a_n-a\big|\lt \frac{\eps}{2}\) if we make \(\big|a_n-a\big|\lt \frac{\eps}{2|b|}\text{.}\) But wait! What if \(b=0\text{?}\) We could handle this as a separate case or we can do the following ‚Äúslick trick.‚Äù Notice that we can add one more line to the above string of inequalities: \(\left|a_n\right|\left|b_n-b\right|+\left|b\right|\left|a_n-a\right|\lt \left|a_n \right|\left|b_n-b\right|+\left(\left|b\right|+1\right)\left|a_n-a \right|\text{.}\) Now we can make \(\left|a_n-a\right|\lt \frac{\eps}{2\left(|b|+1\right)}\) and not worry about dividing by zero.
    Making \(\big|a_n\big|\big|b_n-b\big|\lt \frac{\eps}{2}\) requires a bit more finesse. At first glance, one would be tempted to try and make \(\big|b_n-b\big|\lt \frac{\eps}{2|a_n|}\text{.}\) Even if we ignore the fact that we could be dividing by zero (which we could handle), we have a bigger problem. According to the definition of \(\lim_{n\rightarrow\infty}b_n=b\text{,}\) we can make \(\big|b_n-b\big|\) smaller than any given fixed positive number, as long as we make \(n\) large enough (larger than some \(N\) which goes with a given epsilon). Unfortunately, \(\frac{\eps}{2|a_n|}\) is not fixed as it has the variable \(n\) in it; there is no reason to believe that a single \(N\) will work with all of these simultaneously. To handle this impasse, we need the following:

    Lemma 7.2.7.

    A Convergent Sequence Is Bounded.
    If \(\lim_{n\rightarrow\infty}a_n=a\text{,}\) then there exists \(B>0\) such that \(|a_n|\leq B\) for all \(n\text{.}\)

    Problem 7.2.8.

    Prove Lemma¬†7.2.7.
    Hint.
    We know that there exists \(N\) such that if \(n>N\text{,}\) then \(\abs{a_n-a}\lt 1\text{.}\) Let \(B=\)max\(\left(\abs{a_1},\abs{a_2},\ldots,\abs{a_{\lceil{N}\rceil}},\abs{a}+1\right)\text{,}\) where \(\lceil{N}\rceil\) represents the smallest integer greater than or equal to \(N\text{.}\) Also, notice that this is not a convergence proof so it is not safe to think of \(N\) as a large number.
    Actually, this is a dangerous habit to fall into even in convergence proofs.
    Armed with this bound \(B\text{,}\) we can add on one more inequality to the above scrapwork to get
    \begin{align*} \left|a_n\cdot b_n-a\cdot b\right|\amp =\left|a_n\cdot b_n-a_n\cdot b+a_n\cdot b-a\cdot b\right|\\ \amp \leq\left|a_n\cdot b_n-a_n\cdot b\right|+\left|a_n\cdot b-a\cdot b\right|\\ \amp =\left|a_n\right|\left|b_n-b\right|+\left|b\right|\left|a_n-a\right|\\ \amp \lt B\left|b_n-b\right|+\left(\left|b\right|+1\right)\left|a_n-a\right| \end{align*}
    At this point, we should be able to make the last line of this less than \(\eps\text{.}\)
    END OF SCRAPWORK

    Problem 7.2.9.

    Prove Theorem¬†7.2.6.

    Corollary 7.2.10.

    (Corollary to Theorem¬†7.2.6) If \(\displaystyle\lim_{n\rightarrow\infty}a_n=a\) and \(c\in\mathbb{R}\text{,}\) then \(\displaystyle\lim_{n\rightarrow\infty}c\cdot a_n=c\cdot a\text{.}\)

    Problem 7.2.11.

    Prove Corollary¬†7.2.10.
    Just as Theorem¬†7.2.6 says that the limit of a product is the product of the limits, we can prove the analogue for quotients.

    Theorem 7.2.12.

    Suppose \(\displaystyle\lim_{n\rightarrow\infty}a_n=a\) and \(\displaystyle\lim_{n\rightarrow\infty}b_n=b\text{.}\) Also suppose \(b\neq 0\) and \(b_n\neq 0,\forall\,n\text{.}\) Then \(\displaystyle\lim_{n\rightarrow\infty}\left(\frac{a_n}{b_n}\right)=\frac{a}{b}\text{.}\)

    Problem 7.2.13.

    Prove Theorem¬†7.2.12.
    SCRAPWORK
    To prove this, let‚Äôs look at the special case of trying to prove \(\lim_{n\rightarrow\infty}\left(\frac{1}{b_n}\right)=\frac{1}{b}\text{.}\) The general case will follow from this and Theorem¬†7.2.6. Consider \(\big|\frac{1}{b_n}-\frac{1}{b}\big|=\frac{|b-b_n|}{|b_n||b|}\text{.}\) We are faced with the same dilemma as before; we need to get \(\big|\frac{1}{b_n}\big|\) bounded above. This means we need to get \(|b_n|\) bounded away from zero (at least for large enough \(n\)).
    This can be done as follows. Since \(b\neq 0\text{,}\) then \(\frac{|b|}{2}>0\text{.}\) Thus, by the definition of \(\lim_{n\rightarrow\infty}b_n=b\) , there exists \(N_1\) such that if \(n>N_1\text{,}\) then \(|b\mathopen|-|b_n|\leq\big|b-b_n\big|\lt \frac{|b|}{2}\text{.}\) Thus when \(n>N_1\text{,}\) \(\frac{|b|}{2}\lt |b_n|\) and so \(\frac{1}{|b_n|}\lt \frac{2}{|b|}\text{.}\) This says that for \(n>N_1\text{,}\) \(\frac{|b-b_n|}{|b_n||b|}\lt \frac{2}{|b|^2}|b-b_n|\text{.}\) We should be able to make this smaller than a given \(\eps>0\text{,}\) provided we make \(n\) large enough.
    These theorems allow us to compute limits of complicated sequences and rigorously verify that these are, in fact, the correct limits without resorting to the definition of a limit.

    Problem 7.2.14.

    Identify all of the theorems implicitly used to show that
    \begin{equation*} \lim_{n\rightarrow\infty}\frac{3n^3-100n+1}{5n^3+4n^2-7}=\lim_{n \rightarrow\infty}\frac{n^3\left(3-\frac{100}{n^2}+\frac{1}{n^3}\right)}{n^3 \left(5+\frac{4}{n}-\frac{7}{n^3}\right)}=\frac{3}{5}\text{.} \end{equation*}
    Notice that this presumes that all of the individual limits exist. This will become evident as the limit is decomposed.
    There is one more tool that will prove to be valuable.

    Theorem 7.2.15.

    Squeeze Theorem for Sequences
    Let \(\left(r_n\right),\left(s_n\right)\text{,}\) and \(\left(t_n\right)\) be sequences of real numbers with \(r_n\leq s_n\leq t_n,\forall\) positive integers \(n\text{.}\) Suppose \(\lim_{n\rightarrow\infty}r_n=s=\lim_{n\rightarrow\infty}t_n\text{.}\) Then \(\left(s_n\right)\) must converge and \(\lim_{n\rightarrow\infty}s_n=s\text{.}\)

    Problem 7.2.16.

    Prove Theorem¬†7.2.15.
    Hint.
    This is probably a place where you would want to use \(s-\eps\lt s_n\lt s+\eps\) instead of \(|s_n-s|\lt \eps\text{.}\)
    The Squeeze Theorem holds even if \(r_n\leq s_n\leq t_n\) holds for only sufficiently large \(n\text{;}\) i.e., for \(n\) larger than some fixed \(N_0\text{.}\) This is true because when you find an \(N_1\) that works in the original proof, this can be modified by choosing \(N=\)max\(\left(N_0,N_1\right)\text{.}\) Also note that this theorem really says two things: \(\left(s_n\right)\) converges and it converges to \(s\text{.}\) This subtle point affects how one should properly use the Squeeze Theorem.

    Example 7.2.17.

    Prove \(\displaystyle\lim_{n\rightarrow\infty}\frac{n+1}{n^2}=0\text{.}\)

    Proof.

    Notice that \(0\leq\frac{n+1}{n^2}\leq\frac{n+n}{n^2}=\frac{2}{n}\text{.}\) Since \(\displaystyle\lim_{n\rightarrow\infty}0=0=\lim_{n\rightarrow\infty}\frac{2}{n}\text{,}\) then by the Squeeze Theorem, \(\displaystyle\lim_{n\rightarrow\infty}\frac{n+1}{n^2}=0\text{.}\)
    Notice that this proof is completely rigorous. Also notice that this is the proper way to use the Squeeze Theorem. Here is an example of an {improper} use of the Squeeze Theorem.
    How not to prove Example¬†7.2.17. Notice that
    \begin{equation*} 0\leq\frac{n+1}{n^2}\leq\frac{n+n}{n^2}=\frac{2}{n}\text{,} \end{equation*}
    so
    \begin{equation*} 0=\lim_{n\rightarrow\infty}0 \leq \lim_{n\rightarrow\infty}\frac{n+1}{n^2}\leq\lim_{n\rightarrow\infty}\frac{2}{n}=0 \end{equation*}
    and
    \begin{equation*} \lim_{n\rightarrow\infty}\frac{n+1}{n^2}=0\text{.} \end{equation*}
    This is incorrect in form because it presumes that \(\lim_{n\rightarrow\infty}\frac{n+1}{n^2}\) exists, which we don‚Äôt yet know. If we knew that the limit existed to begin with, then this would be fine. The Squeeze Theorem proves that the limit does in fact exist, but it must be so stated.
    These general theorems will allow us to rigorously explore convergence of power series in the next chapter without having to appeal directly to the definition of convergence. However, you should remember that we used the definition to prove these results and there will be times when we will need to apply the definition directly. However, before we go into that, let‚Äôs examine divergence a bit more closely.
