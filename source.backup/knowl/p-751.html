This definition also makes very precise what we mean when we say that \(f(x)\) should be “close to” \(f(a)\) whenever \(x\) is “close enough to” \(a\text{.}\) For example, intuitively we know that \(f(x)=x^2\) should be continuous at \(x=2\text{.}\) This means that we should be able to get \(x^2\) to within, say, \(\eps=.1\) of \(4\) provided we make \(x\) close enough to \(2\text{.}\) Specifically, we want \(3.9\lt x^2\lt 4.1\text{.}\) This happens exactly when \(\sqrt{3.9}\lt x\lt \sqrt{4.1}\text{.}\) Using the fact that \(\sqrt{3.9}\lt 1.98\) and \(2.02\lt \sqrt{4.1}\text{,}\) then we can see that if we get \(x\) to within \(\delta=.02\) of \(2\text{,}\) then \(\sqrt{3.9}\lt 1.98\lt x\lt 2.02\lt \sqrt{4.1}\) and so \(x^2\) will be within .\(1\) of \(\,4\text{.}\) This is very straightforward. What makes this situation more difficult is that we must be able to do this for any \(\eps>0\text{.}\)
