  1. Clearly

    \begin{equation*} \lim_{n\rightarrow\infty}s_n=s \text{ if and only if } \lim_{n\rightarrow\infty}\left(s_n-s\right)=0\text{.} \end{equation*}

  2. Again notice that this says that we can make \(s_n\) as close to \(s\) as we wish (within \(\eps\)) by making \(n\) large enough (\(>N)\text{.}\) As before, this definition makes these notions very specific.

  3. Notice that \(|s_n-s|\lt \eps\) can be written in the following equivalent forms

    1. \(\displaystyle |s_n-s|\lt \eps\)

    2. \(\displaystyle -\eps\lt s_n-s\lt \eps\)

    3. \(\displaystyle s-\eps\lt s_n\lt s+\eps\)

    4. \(\displaystyle s_n\in\left(s-\eps,s+\eps\right)\)

    and we are free to use any one of these which is convenient at the time.
