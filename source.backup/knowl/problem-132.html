Problem 9.4.14.

Show that if \(f\) is differentiable on some interval \((a,b)\) and that \(f^\prime(c)\lt 0\) for some \(c\in (a,b)\) then there is an interval, \(I\subset (a,b)\text{,}\) containing \(c\) such that for every \(x, y\) in \(I\) where \(x\ge y\text{,}\) \(f(x)\le f(y)\text{.}\)
