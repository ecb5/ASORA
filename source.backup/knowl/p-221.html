Returning to the Brachistochrone problem we observe that

\begin{align} \frac{\sin\alpha}{v} \amp = c\notag\\ \end{align}

and since \(\sin\alpha = \frac{d x}{d s}\) we see that

\begin{align} \frac{\frac{d x}{d s}}{\sqrt{2gy}} \amp = c\notag\\ \frac{d x}{\sqrt{2gy(ds)^2}} \amp = c\notag\\ \frac{d x}{\sqrt{2gy\left[(d x)^2+(d y)^2\right]}} \amp = c\text{.}\tag{4.1.5} \end{align}
