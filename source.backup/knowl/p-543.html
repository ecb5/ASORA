We can make this whole thing less than \(\eps\text{,}\) provided we make each term in the sum less than \(\frac{\eps}{2}\text{.}\) We can make \(\big|b\big|\big|a_n-a\big|\lt \frac{\eps}{2}\) if we make \(\big|a_n-a\big|\lt \frac{\eps}{2|b|}\text{.}\) But wait! What if \(b=0\text{?}\) We could handle this as a separate case or we can do the following “slick trick.” Notice that we can add one more line to the above string of inequalities: \(\left|a_n\right|\left|b_n-b\right|+\left|b\right|\left|a_n-a\right|\lt \left|a_n \right|\left|b_n-b\right|+\left(\left|b\right|+1\right)\left|a_n-a \right|\text{.}\) Now we can make \(\left|a_n-a\right|\lt \frac{\eps}{2\left(|b|+1\right)}\) and not worry about dividing by zero.
