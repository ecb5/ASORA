Definition 7.3.3.

A sequence, \(\left(a_n\right)_{n=1}^\infty\text{,}\) diverges to positive infinity if for every real number \(r\text{,}\) there is a real number \(N\) such that \(n>N\imp a_n>r\text{.}\)

A sequence, \(\left(a_n\right)_{n=1}^\infty\text{,}\) diverges to negative infinity if for every real number \(r\text{,}\) there is a real number \(N\) such that \(n>N\imp a_n\lt r\text{.}\)

A sequence is said to diverge to infinity if it diverges to either positive or negative infinity.
